What is new in Slothy Terminal right now

GPU-Accelerated Terminal

Powered by libghostty with Metal rendering. The same terminal engine behind the Ghostty terminal app, with full IME support and native Ghostty config file compatibility.

Background Task Queue

Queue AI prompts for headless execution without occupying a chat tab. Priority scheduling, auto-retry with exponential backoff, live log streaming, and crash recovery built in.

Native Chat + CLI Tabs

Use Claude and OpenCode in native chat tabs or classic CLI tabs in the same window. Jump between modes without breaking flow.

Risky Tool Detection

Background tasks are scanned for dangerous operations like git push, rm -rf, or writing to .env files. Risky tasks pause the queue for your approval.

Production Chat Engine

State-machine architecture with transport and storage layers. Session snapshots persist conversations, usage data, and model selection for instant recovery across restarts.

Ask Mode & Smart Routing

OpenCode Ask Mode instructs the agent to clarify requirements before coding. Smart Claude CLI path resolution prefers native Mach-O binaries over Node.js wrappers.
